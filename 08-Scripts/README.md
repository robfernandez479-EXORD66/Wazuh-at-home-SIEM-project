# Scripts

This folder contains helper scripts used for testing detections, generating logs, or automating parts of the Wazuh lab.

---

## 1. Purpose of This Folder

To store scripts that support:

- Detection testing  
- Log generation  
- Automation  
- Data collection  

---

## 2. Typical Files Stored Here

| File | Description |
|------|-------------|
| `generate-encodedcommand.ps1` | PowerShell script to trigger EncodedCommand logs |
| `ssh-fail-loop.sh` | Script to simulate SSH brute force attempts |
| `fim-test.ps1` | Script to modify monitored files for FIM testing |

---

## 3. Notes

All scripts are sanitized and safe for demonstration.  
They are intended for lab use only.

---

## 4. Related Folders

- `04-Data-Samples/` — Logs generated by these scripts
