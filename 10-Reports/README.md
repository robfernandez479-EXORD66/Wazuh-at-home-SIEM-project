# Reports

This folder contains case studies, incident write-ups, and analysis reports based on alerts generated in the Wazuh-at-home SIEM environment.

---

## 1. Purpose of This Folder

To demonstrate:

- Analytical thinking  
- SOC investigation skills  
- Incident documentation  
- MITRE ATT&CK mapping  
- Root cause analysis  

---

## 2. Typical Files Stored Here

| File | Description |
|------|-------------|
| `encodedcommand-incident-report.md` | Full write-up of a PowerShell alert investigation |
| `ssh-bruteforce-case-study.md` | Analysis of repeated SSH failures |
| `fim-incident-report.md` | Unauthorized file modification case |

---

## 3. Report Structure

Each report includes:

- Executive summary  
- Timeline  
- Evidence  
- Analysis  
- MITRE mapping  
- Recommendations
