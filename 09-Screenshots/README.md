# Screenshots

This folder contains visual evidence of alerts, dashboards, and investigations performed in the Wazuh-at-home SIEM environment.

---

## 1. Screenshot Naming Convention

Use the following format:

alertname_context_01.png


Examples:

- `encodedcommand_alert_01.png`
- `sshbruteforce_dashboard_02.png`
- `fim_change_rawlog_01.png`

---

## 2. Screenshot Categories

- **Alert views**  
- **Dashboard views**  
- **Raw log views**  
- **Investigation steps**  

---

## 3. Related Folders

- `06-Detections/` — Rules that generated these alerts  
- `05-Dashboards/` — Dashboards shown in screenshots
