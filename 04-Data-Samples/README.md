# Data Samples

This folder contains sample logs, alerts, and event data generated during testing of the Wazuh-at-home SIEM environment.

---

## 1. Purpose of This Folder

To provide real examples of:

- Raw Windows event logs  
- Sysmon logs  
- Wazuh alert JSON  
- Authentication failures  
- PowerShell events  
- FIM (File Integrity Monitoring) events  

These samples help demonstrate how detections were validated and how alerts appear in Wazuh.

---

## 2. Typical Files Stored Here

| File | Description |
|------|-------------|
| `encodedcommand-alert.json` | Alert generated by PowerShell EncodedCommand test |
| `ssh-bruteforce-sample.json` | Sample Linux authentication failure logs |
| `fim-change.json` | File Integrity Monitoring alert sample |
| `sysmon-process-create.json` | Sysmon event for process creation |

---

## 3. How to Use These Samples

- Reviewers can see exactly what logs triggered detections  
- Analysts can compare raw logs to detection rules  
- Useful for training, documentation, and portfolio demonstrations
