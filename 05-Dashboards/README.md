# Dashboards

This folder contains screenshots and exports of Wazuh dashboards used to visualize alerts, endpoint activity, and detection performance.

---

## 1. Purpose of This Folder

To demonstrate:

- Custom dashboards created for monitoring  
- Visualizations of detection activity  
- SOC-style views for triage and investigation  

---

## 2. Typical Files Stored Here

| File | Description |
|------|-------------|
| `encodedcommand-dashboard.png` | Dashboard showing PowerShell EncodedCommand alerts |
| `ssh-bruteforce-dashboard.png` | Authentication failure visualization |
| `fim-dashboard.png` | File Integrity Monitoring dashboard |

---

## 3. Dashboard Categories

- **Detection dashboards**  
- **Endpoint activity dashboards**  
- **Authentication dashboards**  
- **Sysmon visibility dashboards**
